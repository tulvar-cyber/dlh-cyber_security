## Network Traffic Monitoring & Analysis
This project-based module explores how network traffic is captured, filtered, and analyzed in a cybersecurity context
## 0. Basic Packet Analysis
Download the provided PCAP file and analyze the network traffic to identify the protocols used, the source IP address, and the HTTP destination port.

## 1. HTTP Traffic Analysis
Download the provided PCAP file and analyze the HTTP packets to extract authentication information.

## 2. DNS Query Analysis
Download the provided PCAP file and analyze DNS queries and responses to identify suspicious domain activity.

## 3. TCP Connection Analysis
Download the provided PCAP file and analyze a complete TCP connection to understand the three-way handshake, data transfer, and connection termination.

## 4. Advanced Filtering
Download the provided PCAP file and use advanced filtering techniques to extract FTP credentials from mixed network traffic containing multiple protocols.

## 5. TCP Stream Reconstruction
Download the provided PCAP file and reconstruct a fragmented TCP stream to extract a complete secret message that was split across multiple packets.

## 6. Malicious Traffic Detection
Download the provided PCAP file and analyze network traffic to detect malicious patterns including beaconing behavior and data exfiltration.

## 7. RDP Attack Analysis
Download the provided PCAP file and analyze the RDP traffic using tcpdump to extract authentication information and identify attack patterns.
