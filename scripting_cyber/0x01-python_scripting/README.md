Python for Cybersecurity

0. Get IP Address (IPv4 DNS Resolution)
Create a Python script that resolves a domain name to its IPv4 address using the socket library.

1. Get All DNS Records
Create a Python script that queries and displays all DNS record types using the dnspython library.

2. Download a Web Page
Create a Python script that downloads and saves a web page's HTML content using the requests library.

3. Get HTTP Headers
Create a Python script that retrieves and displays HTTP response headers from a website.

4. Check All Pages (Web Crawler)
Create a Python script that recursively crawls a website and discovers internal links up to a specified depth.

5. Check One Port (Port Scanner)
Create a Python script that checks if a specific port is open on a host.
